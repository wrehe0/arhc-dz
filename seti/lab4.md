
Лабораторная работа подключение к Raspberry PI по протоколу VNC
Для подключения к Raspberry Pi вам понадобится следующее:

ваш Raspberry Pi и устройство с клиентом VNC, подключенные к одной сети (например, домашней беспроводной сети или VPN)

имя хоста или IP-адрес вашего Raspberry Pi

действительное имя пользователя и пароль для учетной записи на вашем Raspberry Pi

Если вы не знаете IP-адрес своего устройства, воспользуйтесь нашей инструкцией по определению IP-адреса.

Скачайте TigerVNC. Вы можете установить последнюю версию с страницы релизов в репозитории GitHub. Нажмите на ссылку в последнем релизе и найдите бинарный файл exe; пользователям Linux следует установить jar.
На клиентском устройстве запустите TigerVNC. Windows можно запустить бинарный файл двойным щелчком мыши. В Linux установите Java с помощью sudo apt install default-jre, затем запустите java -jar VncViewer-.jar, заменив на версию, которую вы скачали.

В поле «VNC-сервер» введите IP-адрес вашего Raspberry Pi. 192.168.129.150



Нажмите кнопку «Параметры». Перейдите на вкладку «Ввод». Установите флажок «Показывать точку при отсутствии курсора», чтобы в TigerVNC всегда был виден курсор.



если TigerVNC выдает предупреждение, нажмите кнопку «Да», чтобы разрешить доступ к Raspberry Pi.

Введите в окно имя пользователя user и пароль 123

справка
Чтобы скопировать файл в определенную папку, укажите путь к ней после : в команде scp. Создайте папку перед выполнением scp, так как scp не создает папки автоматически. Например, следующая команда копирует файл с именем myfile.txt в папку project/ в домашней папке пользователя:
scp myfile.txt <username>@<pi_ip_address>:project/

Скопируйте файлы с вашего Raspberry Pi
скопируйте файл по номеру своего варианта из каталога letunov/ к себе на компьютер и сохрание его в тепозитории отчета к лабораторной работе. в отчете сделайте скриншот данной операции с треем.
Синтаксис команды простой. Это выглядит так:

scp [опции] источник назначение

Например, чтобы скопировать файл с именем myfile.txt из домашнего каталога пользователя на Raspberry Pi в текущий каталог на другом компьютере, выполните следующую команду:

scp <username>@<pi_ip_address>:myfile.txt
