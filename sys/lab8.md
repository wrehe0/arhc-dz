# Практическая работа №8
## Тестирование персонального компьютера 
с помощью утилиты Passmark Perfomance Test

Цель: Провести финальное тестирование стабильности системы для последующей разработки технического задания по внедрению компьютерной системы.

### Теоретические сведения:
PerformanceTest - набор тестов, позволяющих оценить общую производительность вашего ПК по сравнению с другими компьютерами. В программу входят двадцать семь стандартных тестов в семи группах, плюс еще пять для пользовательских тестов.
Среди стандартных можно отметить тесты процессоров, 2D и 3D графики, дисковых накопителей, памяти, CD/DVD-приводов и многих других составляющих компьютера.

 
### В задачу утилиты входит:
• Выяснение того, работает ли ваш ПК на том уровне, на котором он должен работать;
• Сравнение производительности вашего компьютера с производительностью машин подобной и/или более крутой конфигурации;
• Объективные, независимые измерения помогут вам сделать правильное решение при покупке;
• Использование самых последних испытаний для создания ваших собственных сценариев эталонного тестирования

### Стандартные наборы тестов:
• Тестирование ЦП Математические операции: сжатие, шифрование, MMX/SSE, инструкции 3DNow! и др.;
• Тестирование 2D-графики: черчение/рисование, битовые карты/битовые матрицы/побитовое отображение, шрифты, текст и элементы графического пользовательского интерфейса;
• Тестирование 3D-графики: трехмерная графика уровня DirectX 8.1, DirectX 9 и анимация;
• Тестирование диска: чтение, запись и поиск файлов на диске;
• Тестирование памяти: ассигнование и доступ к скорости памяти и оценка эффективности;
• Тестирование CD и DVD: проверка скорости CD/DVD драйверов.
• Усовершенствованные настраиваемые тестирования (доступны только в зарегистрированной версии):
• Усовершенствованный тест для диска;
• Усовершенствованные тесты для CD/DVD;
• Усовершенствованный тест 3D -графики;
• Усовершенствованный тест организации работы в сети (для Ethernet, интернета и беспроводной связи);
• Усовершенствованный тест для памяти;
• Усовершенствованный тест мультипрограммной работы.

## Контрольные вопросы

### 1. Для каких целей может быть использована тестируемая компьютерная система?

Учитывая полученные результаты тестирования (низкий 3D Mark, средний CPU Mark, хороший Disk Mark, отличные температурные показатели), данная компьютерная система может быть использована для следующего круга задач:

- **Офисная работа:** Работа с текстовыми редакторами (Word), электронными таблицами (Excel), базами данных, электронной почтой и web-интерфейсами.
- **Образовательный процесс (базовый уровень):** Изучение информатики на теоретическом уровне, работа в средах программирования начального уровня (Pascal, Python без тяжелых библиотек), просмотр видео-лекций.
- **Мультимедиа (воспроизведение):** Просмотр видео в разрешении до Full HD, воспроизведение музыки.
- **Серверные или терминальные задачи:** Благодаря низкому энергопотреблению и отличной температуре, система может выступать в роли тонкого клиента или файлового сервера в локальной сети.

**Система НЕ подходит** для:
- Современных игр (из-за провального 3D Mark).
- Видеомонтажа и 3D-моделирования.
- Ресурсоемких научных расчетов (из-за слабых математических вычислений Integer Math).

### 2. Какие параметры компьютерной системы можно было бы улучшить? Укажите пример.

Наиболее узким местом системы является графическая подсистема и, в меньшей степени, процессор.

**Пример улучшения конфигурации:**

1.  **Видеоадаптер (самый критичный апгрейд):**
    *   *Текущий:* Встроенная графика (или устаревший дискретный адаптер) с результатом 3D Mark 154.
    *   *Рекомендуемый:* Дискретная видеокарта уровня **NVIDIA GeForce GTX 1050 Ti** или **RX 560** (б/у рынок).
    *   *Эффект:* Позволит запускать современные образовательные программы с 3D-интерфейсом, простые игры для тестирования, а также повысит производительность в работе с графическими редакторами (Photoshop).

2.  **Оперативная память (ОЗУ):**
    *   *Текущий:* По результату теста Available RAM (4669 МБ) видно, что объем составляет около 4-8 ГБ.
    *   *Рекомендуемый:* Увеличение объема до **16 ГБ** (двухканальный режим, если используется один модуль).
    *   *Эффект:* Хотя задержка памяти отличная (27), увеличение объема и пропускной способности (Memory Threaded) позволит комфортно работать с большим количеством вкладок браузера и многозадачностью.
